# Visual QAQC Rules

Do not mark work complete unless the change is visible in the uploaded QC PNG in the correct target detail, plan, section, or sheet viewport. If text is involved, export at sufficient resolution and report whether text is still visible, wrong, or unclear. If the full sheet export is too small, export a focused target view or crop region.

Use visual status values: VISIBLE_CONFIRMED, NOT_VISIBLE_REQUIRES_ADJUSTMENT, WRONG_LOCATION_REQUIRES_ADJUSTMENT, TEXT_STILL_VISIBLE, OCR_UNCLEAR_EXPORT_HIGHER_RESOLUTION.
