# Suggested External RAG Sources

Add downloaded/plain-text notes from these sources to the corpus folder as permitted by their licenses and your firm standards:

- Autodesk Revit API SDK / samples and developer documentation.
- Revit API Docs pages for the Revit versions used by the project.
- RevitLookup notes and examples for inspecting elements, parameters, properties, and relationships.
- RevitPythonShell notes for IronPython shell behavior.
- pyRevit examples for practical Revit automation patterns.
- IDS internal Revit bridge cycle logs, successful scripts, failed scripts, and final corrected patterns.

Keep external documentation summaries concise. The highest-value data is your own successful bridge cycles, because it captures exact Revit version, RPS behavior, coordinate setup, QC folder behavior and project standards.
